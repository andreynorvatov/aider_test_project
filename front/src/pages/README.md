# Pages Directory

This directory contains page-level components (routes).

## Structure
```
pages/
├── Home/
│   ├── HomePage.jsx
│   └── HomePage.scss
├── About/
│   ├── AboutPage.jsx
│   └── AboutPage.scss
├── Auth/
│   ├── LoginPage.jsx
│   └── RegisterPage.jsx
├── Dashboard/
│   └── DashboardPage.jsx
└── NotFound/
    └── NotFoundPage.jsx
```

## Guidelines
- Each page gets its own folder
- Include related styles and tests
- Use consistent naming: `PageName.jsx`

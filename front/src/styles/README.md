# Styles Directory

This directory contains all CSS/SCSS stylesheets for the project.

## Structure
```
styles/
├── base/           # Base styles (reset, typography, variables)
├── components/     # Component-specific styles
├── layouts/        # Layout styles (header, footer, sidebar)
├── pages/          # Page-specific styles
├── utils/          # Utilities, mixins, functions
└── main.scss       # Main entry point
```

## Files
- `variables.scss` - CSS custom properties and SCSS variables
- `mixins.scss` - Reusable SCSS mixins
- `reset.scss` - CSS reset/normalize
- `typography.scss` - Font styles and text utilities

# Constants Directory

This directory contains application-wide constants and configuration values.

## Common Files
- `routes.js` - Route path definitions
- `api.js` - API endpoint URLs
- `colors.js` - Color palette constants
- `breakpoints.js` - Responsive breakpoints
- `errors.js` - Error code mappings
- `config.js` - App configuration

## Guidelines
- Use UPPER_SNAKE_CASE for constant names
- Export as const objects
- Group related constants together

# Hooks Directory

This directory contains custom React hooks (or Vue composables).

## Common Hooks
- `useAuth.js` - Authentication state and methods
- `useFetch.js` - Data fetching with loading/error states
- `useLocalStorage.js` - Local storage persistence
- `useDebounce.js` - Debounced values
- `useToggle.js` - Boolean state toggle
- `useMediaQuery.js` - Responsive breakpoints

## Guidelines
- Prefix all hooks with "use"
- Keep hooks focused on a single responsibility
- Document parameters and return values
